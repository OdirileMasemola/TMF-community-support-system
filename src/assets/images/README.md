# Website Images

This folder stores website images such as hero images, campaign images, logos, and foundation-related images.

Images imported in React components should be placed here.

## Naming examples

Use clear file names like:

- `hero-community-support.jpg`
- `food-support-drive.jpg`
- `education-support.jpg`
- `winter-relief.jpg`
- `tmf-logo.png`

## Hero image

Add your hero image as `hero-community-support.jpg` in this folder. The Home page imports it for the split hero layout.
